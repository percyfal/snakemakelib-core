snakemakelib-core - core functionality of snakemakelib¶

snakemakelib core functionality. Adds modules, functions and utilities for working with Snakemake and snakemake-rules.

Features¶

  1. IOTarget classes. IOTarget classes utilize the power of regular expressions to find input and define output targets.
  2. Application classes. Application classes group targets at the application level.
  3. Application hooks. Hooks makes it easy to add custom functionality, such as plot and annotation functions, to applications.
  4. odo parsers. odo parsers are used for parsing program outputs, thereby making it easy to transform between many data formats
  5. bokeh plots. Customized bokeh plots for making interactive graphics
