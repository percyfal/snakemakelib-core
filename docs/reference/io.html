Make targets¶

make_targets(tgt_re, samples)[source]¶

Make targets

Create target names based on the target regular expression.

Parameters:
  • tgt_re (IOTarget) – IOTarget object corresponding to the target regular expression
  • samples (list) – list of dicts where each dict is an annotated sample. The keys correspond to read group labels.
Returns:

targets – list of target names
Return type:

list
