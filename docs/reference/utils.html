find_files(regex, path='.', search=False, limit=None, use_full_path=False)[source]¶

Find files in path that comply with a regular expression.

Parameters:
  • regex (IOTarget | str) – regular expression object of class IOTarget or str
  • path (str) – path to search
  • search (bool) – use re.search instead of re.match for pattern matching
  • limit (dict) – dictionary where keys correspond to regular expression grouping labels and values are lists that limit the returned pattern
  • use_full_path (bool) – use full path in match/search
Returns:

list of file names, prepended with root path
Return type:

flist

Example

The following code block would recursively search for files with suffix ‘.fastq.gz’ from the current directory.

f = find_files(regex="\w+.fastq.gz")

By adding the search parameter re.search is used instead of re.match. In this case, the following code would find files with suffix ‘.fastq.+’:

f = find_files(regex="\w+.fastq.gz", search=True)
